# 개요

이 영상은 **비개발자도 쉽게 AI 에이전트**를 만들 수 있는 **오토메이션 툴 "[[n8n (Node for Node)]]"** 의 사용법을 소개하는 실전 튜토리얼입니다.  
AI 에이전트의 개념을 설명하고, Google Calendar, Gmail, Google Sheets 등 외부 서비스와 연결해 일정 관리 비서를 직접 만드는 과정을 실습 위주로 다룹니다.
# 내용

### AI 자동화 방식 비교  
([00:00](https://www.youtube.com/watch?v=VlsgHC5xkPA&t=0s))  
- **AI 노드 기반 워크플로우**  
  - 미리 정해진 절차대로 실행  
  - 예: PDF → 요약 → 슬랙 전송  
  - 안정적이지만 유연성 부족  

- **AI 에이전트 기반 워크플로우**  
  - AI가 상황에 따라 작업 순서를 스스로 판단  
  - 툴들을 조합해 다양한 요청 처리  
  - 유연성은 뛰어나나 정확성은 모델에 따라 다름

---

### n8n에서 AI 에이전트 만들기 실습  
([03:15](https://www.youtube.com/watch?v=VlsgHC5xkPA&t=195s))  
- **목표**: 일정 조회, 추가/삭제, 참석자 초대, 리마인더 메일 전송  
- **주요 구성 요소**:
  - **트리거**: `On Chat Message` → 채팅 기반 명령 수신  
  - **두뇌 (LLM)**: ChatGPT, Claude, Gemini 등 연결 (API Key 필요)  
  - **메모리**: 최근 대화 문맥 기억용 (`Simple Memory`)  
  - **툴**:
    - Google Calendar (일정 확인, 생성, 삭제)
    - Google Sheets (팀원 목록 조회)
    - Gmail (이메일 발송)

---

### API 연동 절차  
([09:53](https://www.youtube.com/watch?v=VlsgHC5xkPA&t=593s))  
- **Google Cloud Console 설정**
  - 프로젝트 생성
  - API 사용 설정 (Calendar, Sheets, Gmail, Drive)
  - OAuth 동의화면 설정
  - Client ID, Secret 발급
  - 권한(Scope) 부여: 캘린더 전체 접근, 시트 조회/편집, 이메일 발송 등

---

### 프롬프트 설정 및 테스트  
([18:36](https://www.youtube.com/watch?v=VlsgHC5xkPA&t=1116s))  
- **시스템 프롬프트 작성**  
  - 툴 리스트 및 사용 목적 설명 포함  
  - 오늘 날짜 자동 삽입: `${now:YYYY-MM-DD}`  
  - 타임존 `Asia/Seoul`로 설정  

- **테스트 예시**  
  - “내일 일정 브리핑해 줘” → 캘린더 조회  
  - “저녁 7시에 미팅 추가해 줘” → 참석자 시트 조회 + 일정 생성  
  - “리마인더 이메일 보내줘” → Gmail 툴 활용

---

### Web 채팅 인터페이스 만들기  
([22:17](https://www.youtube.com/watch?v=VlsgHC5xkPA&t=1337s))  
- **n8n Webchat URL 생성**  
  - 로그인 기반 접근 설정 가능  
  - 사용자 정의 인사말 설정 (예: "자비스입니다. 무엇을 도와드릴까요?")  
  - URL 접속 시 챗봇처럼 바로 대화 가능  
  - 일정 변경도 실시간 반영 예시 제공

# 결론  
n8n은 **코딩 없이도 AI 기반 챗봇/에이전트를 구축할 수 있는 강력한 도구**입니다.  
복잡한 로직을 작성할 필요 없이 필요한 툴 몇 개만 연결하면 유연하게 업무를 처리할 수 있습니다.  
다만 툴이 많아지면 정확도 저하 가능성도 있으므로, **기본 기능부터 시작해 점차 확장**하는 것을 권장합니다.
