# 개요

생성형 AI의 빠른 성장과 비즈니스 도입 현황을 강조하며, Oracle APEX와 같은 로우코드(Low-code) 플랫폼을 통해 AI 도입의 장벽을 낮추고, 비즈니스 전문가들도 쉽게 애플리케이션을 개발할 수 있게 되는 이점을 설명함.

 APEX는 AI 중심의 기능을 통합하여 데이터 모델링, 애플리케이션 생성, 디버깅 등 다양한 개발 작업을 자동화하고, UI 생성과 데이터 시각화까지 지원합니다. 이로 인해 기업들은 더 빠르고 효율적으로 AI 기반의 비즈니스 애플리케이션을 개발할 수 있을 것
 
# 내용

## 1. 생성형 AI의 성장과 도입 (00:00 - 01:04)
- 생성형 AI는 2022년 하반기 ChatGPT의 출시 이후 빠르게 성장하고 있음.
- 2026년까지 비즈니스에서 75% 이상 생성형 AI를 도입할 것으로 예상.
- AI 도입은 비즈니스 혁신을 촉진하지만, 초기 도입 시 과제가 있을 수 있음.

## 2. AI 도입의 과제와 로우코드 해결 방안 (01:04 - 03:01)
- AI 도입 시 비즈니스는 AI의 가치 이해 부족, 기술적 조정 문제, 인재 부족 등의 과제를 마주함.
- 로우코드 플랫폼은 비즈니스 전문가들이 애플리케이션을 쉽게 개발할 수 있게 해줌.
- Oracle APEX와 같은 로우코드 플랫폼은 AI 도입의 장벽을 낮추는 데 중요한 역할을 할 수 있음.

## 3. 로우코드 플랫폼의 장점 (03:01 - 04:44)
- 로우코드 플랫폼은 비즈니스 애플리케이션을 빠르고 효율적으로 개발할 수 있도록 도와줌.
- APEX는 비즈니스 전문가도 애플리케이션 개발에 참여할 수 있게 하여 개발 속도와 생산성을 향상시킴.
- 로우코드 플랫폼은 프로토타입 제작과 검증을 쉽게 할 수 있어 유연성과 빠른 실행을 지원.

## 4. APEX의 역사와 인기 (04:44 - 06:25)
- APEX는 2000년에 출시되어 25년 동안 발전해 온 오라클의 로우코드 플랫폼.
- 현재 전 세계에서 2,100만 개 이상의 애플리케이션이 APEX를 통해 만들어졌음.
- APEX는 보안 강화와 확장성을 기반으로 엔터프라이즈 애플리케이션 개발에 적합한 솔루션을 제공.

## 5. [[Oracle APEX의 핵심 기능과 구조]] (06:25 - 07:59)
- APEX는 AI 중심의 기능을 통해 데이터를 자연어로 처리하고 SQL 쿼리 생성 및 디버깅을 지원.
- APEX는 클라우드 및 온프레미스 환경 모두에서 사용 가능.
- AI와 APEX의 통합을 통해 비즈니스 애플리케이션에 AI를 쉽게 적용할 수 있음.

## 6. AI 통합과 데이터 모델링 시연 (07:59 - 10:30)
- APEX는 AI를 활용하여 데이터 모델링을 자연어로 설명하고 자동으로 SQL 쿼리를 생성.
- 이 기능을 통해 개발자는 데이터베이스 설계에 대한 깊은 지식 없이도 쉽게 데이터베이스를 구축할 수 있음.
- APEX는 데이터 모델링 작업을 자동화하여 개발 시간을 단축시킴.

## 7. UI 생성 및 애플리케이션 개발 ([10:30 - 15:06](https://youtu.be/sy-I1x8Nvlw?si=9V-IOUkHhzKhLLU0&t=679))
- APEX는 AI가 사용자가 요구하는 UI를 자동으로 생성할 수 있도록 돕습니다.
- AI 도우미는 애플리케이션의 구조를 자동으로 설계하고, UI가 다양한 디바이스에서 원활히 동작하도록 지원.
- 복잡한 코딩 없이도 사용자 요구를 반영한 애플리케이션을 빠르게 구현할 수 있음.
- 구성된 데이터는 ERD 다이어그램으로 확인 가능

## 8. AI를 활용한 지도 및 데이터 확장 ([15:06 - 19:34](https://youtu.be/sy-I1x8Nvlw?si=YhLa7_srMPXUsuiz&t=1133))
- APEX에서 AI를 활용하여 애플리케이션에 지도 기능을 추가할 수 있음.
- 사용자는 지도에 데이터를 시각화하여 직관적으로 정보를 전달할 수 있음.
- AI는 데이터를 자동으로 처리하고, 시각화하여 애플리케이션을 확장하는 데 도움을 줌.

## 9. AI 대화형 기능과 RAG 기술 (19:34 - 23:58)
- APEX는 AI 대화형 기능을 통해 사용자가 자연어로 질문하고 비즈니스 데이터를 기반으로 답을 받을 수 있게 해줌.
- RAG(반환 증강 생성) 기술을 사용하여, AI는 비즈니스 데이터를 검색하고 정확한 정보를 실시간으로 제공함.
- 이 기능을 통해 사용자는 데이터 기반의 즉각적인 피드백을 받을 수 있음.

## 10. AI 대화창과 비즈니스 데이터 활용 (23:58 - 29:59)
- APEX는 AI 대화창을 통해 사용자가 비즈니스 데이터를 실시간으로 질의하고 답을 받을 수 있도록 해줌.
- AI 대화창은 비즈니스 데이터를 바탕으로 정확한 답변을 제공하고, 데이터 접근을 직관적으로 만듬.
- 이 기능은 실시간 의사결정을 지원하며, 비즈니스 데이터 활용도를 높이는 데 중요한 역할을 함.
