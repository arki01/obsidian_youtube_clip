## 감정일기란?
: 감정일기는 하루 동안의 감정을 1~3줄 정도로 짧게 기록하며,  자신의 감정 상태를 **인식 → 명명 → 정리**하는 습관을 말한다.

## 감정일기의 과학적 효과

- **감정 명명(labeling)**  
  → 뇌의 편도체 활동 감소 → 스트레스, 불안 완화  
  (출처: UCLA 심리학 연구, Matthew D. Lieberman 외)

- **감정 정리 글쓰기**  
  → 우울감, 무기력 감소 + 자기효능감 증가  
  (출처: James Pennebaker, 감정표현 글쓰기 연구)

- **감정 기록의 반복성**  
  → 실행력 향상, 감정 탄력성(Resilience) 향상  
  → 감정 조절력 + 통찰력 증진

## 감정일기 작성법

### 효과가 떨어지는 작성법

| 문제         | 설명                                   |
| ---------- | ------------------------------------ |
| 감정 폭발에만 집중 | “짜증나서 못 참겠다” 식의 감정 배출 → 오히려 각인 심화 가능 |
| 고립된 이벤트 기록 | 사건 중심 서술 → 감정 흐름과 연결되지 않음            |
| 간헐적 기록     | 2-3달에 한 번씩 → 인지-감정 사이 연결성 약화         |

### 효과적인 감정일기 작성법

#### 1. **짧고 자주 (2~3줄이면 충분)**  
- ✔ 오늘의 기분: "약간 무기력했지만 산책 후엔 조금 나아짐"  
- ✔ 원인 추정: "업무에서 말 없이 넘어간 피드백 때문"

#### 2. **감정 → 관찰 → 행동 연결**  
- 기분 상태를 **판단 없이 받아들이고**  
- “오늘 내가 할 수 있는 작은 행동 1가지”로 마무리

#### 3. **시간대는 자기 전 or 퇴근 직후가 이상적**  
- 하루의 감정을 정리하고 다음 날에 영향을 덜 주게 됨


## 🧾 감정일기 템플릿 예시

```markdown
## 🪞오늘의 감정 일기 (YYYY-MM-DD)

- 오늘의 기분은?  
  → 예: "일이 많아 지쳤지만, 산책 후에 맘이 좀 풀렸음."

- 감정의 원인은?  
  → 예: "회의 중 책임만 강조된 게 부담스러웠던 듯."

- 오늘 나를 위한 한 가지 행동:  
  → 예: "내일은 말로 감정 표현 1번 시도해보기"
```
