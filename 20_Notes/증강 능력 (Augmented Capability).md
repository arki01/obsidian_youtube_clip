# 정의
**증강 능력(Augmented Capability)**은 AI를 단순한 도구가 아닌 ‘협업 파트너’로 인식하고, 이를 통해 자신의 업무 효율, 문제 해결 능력, 창의력, 의사결정력을 증대시키는 총체적 능력이다.

> AI와 함께 일하며 인간의 고유한 사고력, 목적 설정, 피드백 역량을 결합해 생산성과 성과를 증폭시키는 능력

# 내용

## 주요 특징 및 구성 요소

| 구성 요소           | 설명 |
|----------------------|------|
| **AI 인식 태도**     | AI를 검색/자동화 도구가 아닌 ‘협업자’로 인식하는 태도 |
| **프롬프트 대화력**  | AI와 반복적 피드백 기반의 소통으로 정교한 결과 도출 |
| **자기 업무 정의력** | 일의 목적과 맥락을 스스로 정의하고 AI에게 전달하는 능력 |
| **인터랙션 능력**    | AI에게 명확한 명령을 주고 결과에 대해 수정·재질문하는 능력 |
| **맥락 설정력**      | 목적과 배경을 AI에게 명확히 전달하여 더 나은 결과를 유도하는 능력 |

---

## 이중학 교수가 강조한 배경

- GPT-4는 **IQ 157** 수준의 인지 능력 → 인간 상위 0.01%
- 단순한 정보력이나 문제 해결력만으로는 AI와 경쟁 불가
- 인간은 **AI를 팀처럼 활용하고 가치 창출하는 능력**을 갖춰야 함
- **모든 사람이 Agent Boss가 되어야 하는 시대**

| **증강 능력**         | AI와 협업하고, 인간 고유의 판단·공감·맥락 해석을 결합한 능력 |

## 실제 사례

- **보고서 작성**: 초안은 AI에게 맡기고 인간은 메시지·논리 구조 개선에 집중
- **데이터 분석**: Colab/Gemini에서 분석 자동화, 인간은 인사이트 해석
- **전략 기획**: AI가 데이터 기반 기초 설계 → 인간이 시나리오 조율 및 판단

# 요약
- 증강 능력은 AI 시대의 **생존형 역량**이다.
- AI를 **검색 도구가 아닌 ‘동료’**로 바라보는 태도에서 시작한다.
- 인간만이 할 수 있는 **맥락 설정, 피드백, 공감, 목적 정의**와 AI의 계산력·자동화를 결합해야 한다.
