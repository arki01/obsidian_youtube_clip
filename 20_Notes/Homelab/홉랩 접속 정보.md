# 개요
이 문서는 홉랩 서버에 대한 접속 정보 및 포트 체계를 설명합니다. 
각 서비스별로 사용되는 포트를 명시하고, 외부 및 내부 접속 정보를 제공합니다. 
향후 관리 및 활용에 필요한 요소를 정리하여 사용자가 쉽게 접근할 수 있도록 돕습니다.

# 내용

## 홉랩 포트 체계

| 대역   | 용도            |
| ---- | ------------- |
| 2xxx | 서버 관리 / SSH   |
| 3xxx | AI / UI / 실험용 |
| 5xxx | 자동화 / 워크플로우   |
| 9xxx | 인프라 관리 도구     |
|      |               |
### A. 외부(또는 VPN) 진입 포트 – Ubuntu VM
- 모든 외부 접점은 Ubuntu로 수렴

| 포트          | 서비스           | 이유       |
| ----------- | ------------- | -------- |
| 2222        | SSH           | 표준 22 회피 |
| 3000        | OpenWebUI     | AI UI    |
| 5001        | n8n           | 자동화      |
| 9000        | Portainer     | 컨테이너 관리  |
| (추후) 80/443 | Reverse Proxy | 외부 서비스   |



### B. 내부 전용 포트 – 관리 / 도구

|대상|포트|접근 방식|
|---|---|---|
|Proxmox|8006|VPN 내부|
|Windows VM|3389|VPN 내부|
|Ubuntu 내부 서비스|기본값|로컬/VPN|
