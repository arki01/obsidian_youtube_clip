# 개요
여러가지 빅데이터 분석 방법론 중에서 가장 일반적으로 쓰이는 방법론이다. 해당 방법론으로 S.CON의 Analysis 모듈을 개발하는데 좋은 뒷받침이 될 것이라 생각되어 내용을 정리해본다.

# 내용
총 5단계가 있으며 아래 같이 요약해볼 수 있겠다. 
- (1단계) 분석 기획 : 비즈니스 범위 설정 및 프로젝트 정의 (SOW, WBS 작성)
- (2단계) 데이터 준비 : 필요 데이터를 정의하고 데이터 스토어를 설계, 데이터 수집 및 정합성을 점검한다.
- (3단계) 데이터 분석 : 데이터 분석을 통하여 데이터 모델링 후 모델 평가 및 검증을 진행하는 단계
- (4단계) 시스템 구현 : 위 모듈이 정상적으로 작동한다면 시스템으로 구현하여 자동화하고 운영에 착수한다.
- (5단계) 평가 및 전개 : 해당 모델에 대한 평가 이후 성과 및 발전계획을 수립하고 프로젝트를 종료한다.
