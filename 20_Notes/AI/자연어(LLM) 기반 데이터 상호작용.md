# 개요

대형 언어 모델(LLM)을 이용하여 SQL 등 기술 지식 없이도 자연어로 데이터 질의 및 분석이 가능하게 함.

# 내용

## 맥락 및 배경
기존 데이터 분석은 SQL 쿼리, BI 도구 등을 통해 일부 전문가 집단만 활용 가능했음. 데이터의 잠재적 가치에 비해 접근성은 매우 제한적이었음.

## 문제의식
- 기술 문외한에게 데이터 분석은 높은 진입장벽
- 실시간 의사결정이 필요한 현장에서 데이터 접근 속도가 느림
- BI 도구 → 시각화는 가능하나 인사이트 도출까지는 제한

## 기술적 해결 방식
- **LLM 기반 자연어 인터페이스**: 텍스트로 데이터 질의 가능
- **Prompt to SQL 변환**: 사용자의 질의를 자동으로 쿼리화
- **Copilot UI**: 시각적 인터페이스 + LLM 보조 기능

## 활용 예시
- 마케팅: “지난달 신규 가입자 수는?” → 차트 출력
- 제조: “3월 이후 불량률 급등 구간은?” → 기간+원인 분석
- 건설: “최근 1달간 외벽 균열 빈도 추이” → DB 질의 후 시각화
