# 개요

지금 사용하고 있는 노트만 해도 Workflowy, Goodnote, Evernote, Notion 그리고 지금의 Obsidian 이다.
이에 대한 활용처를 생각해보고 가능한 통합해서 사용해보도록 하자. 너무 많은 것들이 분산되어 있으면 메모의 활용이 어려울 수 있기 때문이다.

다만, 한가지 바뀐 점은 이제 Evernote를 배제해야한다는 것이다. 이미 많은 노트들을 쓰고 있었기 때문에 하나 빼는 것도 필요하다. 이제 어떻게 관리할 지 한번 써내려 나가보자.

# 내용

### 2023 Look Back

우선 23년은 생각보다 많은 개인적인 일들이 상당히 많이 있었고, 노트를 써내려갈 여유가 많이 부족했던 한해였던 것 같다. 그래서 일기도 내용이 부족하고 전체적인 메모 수도 적었다.

다만, Notion은 생각보다 적극적으로 잘 썼다. Planning을 하기 위한 여러가지 조건들을 잘 갖추고 있어서 목표 대비하여 하위 Task를 만들어 나가며 구조화 해나가는 노트로서는 어느 노트와도 비교할 수 없을 정도 였다. 이건 24년에도 열심히 사용할 것 같다.

그리고 Goodnote는 거의 사용하지 않았다. 필기를 할 일이 생각보다 별로 없고 정말 개인적인 생각을 천천히 글을 써가며 정리하고 싶을 때 쓰는 정도이지, 이 텍스트들을 다시 타이핑으로 옮기는 별도 과정이 필요한 업무 등에는 잘 사용되지 못하였다. 필기로 사용할 일이 있을 때만 사용할 것 같다.

올해 새로 접하게 된 Obsidian. 이 노트가 아마 Evernote의 일부를 대체해줄 수 있지 않을까 생각된다. 이 노트는 목적에 맞게 나의 Second Brain으로서 지식을 탐구해나가고 구조화해 나가는 것에 아주 유용한 노트라는 생각이 들었다. 작년은 빅데이터분석기사를 공부하기 위해서 사용했다고 하면, 올해는 나의 건강공부([[MOC_근육운동 가이드]]), 경제공부에도 적극 활용할 예정이다.

마지막으로 Workflowy 다. 이건 부정할 수 없을 정도로 잘 쓰고 있다. 빠른 메모가 필요한 회사에서 적극적으로 사용할 수 있을 것 같고, 미팅 메모 등을 빠르게 써려가는데 최적화된 노트이라 생각한다. 메인이기 보다는 서브노트로서 간단한 메모를 구조화해서 저장하고 관리하는데 아주 유용하게 써나갈 수 있을 것 같다.

그리고 이제는 사용하지 않게될 Evernote… 안그래도 점점 작성하는 노트의 수가 줄어들고 있었는데, 유료화로 전환되면서 일년에 노트 몇십개 쓰자고 99,000원을 쓰기엔 너무나도 살인적인 비용인 것 같다는 생각이 들었다. 올해는 할인혜택을 준다고 하지만 향후 충분히 전환할 가능성이 농후해보이기 때문에, 올해는 이제 다른 노트로 넘겨가는 걸 준비해야할 것 같다.
### 앞으로의 노트관리는?

주된 노트는 이제 Notion, Workflowy, Obsidian이 될 것 같다. 기존에 Evernote에서 사용하던 기능들을 Notion과 Obsidian으로 옮기고, 문서와 같은 파일들은 Google Cloud로 전환해야 될 것 같다. 관리하고 싶은 카테고리들은 Workflowy에서 좀 더 디벨롭 시켜보자.

### 노트별 활용방안 정리
- **Workflowy** : Project 들을 확장해나가고 Sprint 단위로 실행해나가도록 구조화 시켜주는 Tool
    - Project List에서 계속해서 확장해나가며, 목표를 성공적으로 수행할 수 있도록 체계화하는 메모 Tool
    - 생각해야하는 가짓수들이 점점 많아지면서 필수적으로 활용해야하는 메모 
- **Goodnote** : Planner를 활용하여 시간단위 목표를 설정하고, 의지를 다잡는 시간을 가지는 Tool
    - Planner를 통하여 Workflowy에서 List-up 하였던 Activity들을 시간단위로 계획하고 수행할 수 있도록 지원
    - Pen으로 작성하며 Digital적으로 정리하였던 것들을 꼼꼼히 되짚어볼 수 있음
    - 필기가 필요한 메모들을 정리
- **Evernote** : Raw 단위의 Memo들을 쉽게 저장할 수 있는 Tool
    - Raw 데이터의 메모들을 관리하는 공간
    - 빠르게 부담없이 메모할 수 있어, 데이터 축적이 용이하다.
    - 파일도 쉽게 업로드하고 관리할 수 있기 때문에, 클라우드 개념으로도 활용 가능하다.
    - Notion을 적용해보려고 했으나, 역시 나한테는 맞지 않은 툴인듯 하다. 노트랑 태그를 일부 조정해서 지금의 나에 맞춰서 재조정해보자!
- **Notion** : Memo를 체계적으로 저장할 수 있는 Tool
    - Notion은 몇번의 고민이 있었지만 적용 못했던 이유들이 있었는데, 현재 어느정도 많이 개선된 듯 하다.
    - Note 량이 점점 많아지다 보니 구조화가 어느정도 가능한 Notion이 필요해 보이긴 한다.
    - 다만 테스트가 필요한 부분이 있어 고민 중이다.
# 결론
- Workflowy 를 통하여 Roadmap 기반으로 Project를 생성하고, 이에 대한 하위 Activity를 만들어가면서 목표를 구체화 시켜 나가는 메모 Tool
- Goodnote는 Planner를 활용하여 시간단위 목표를 설정하고, 의지를 다잡는 시간을 가지는 Tool
- Evernote로 Raw 단위의 Memo들을 쉽게 저장할 수 있는 Tool
	- 노트들을 Tag 단위로 대체할 수 있는 프로그램은 아직 존재하지 않는다.
	- Notion도 검토 해봤지만 역시 노트 구조가 불안정하다. (여러개의 DB가 혼재되어있고 이를 다시 병합하여 운용하는데 어려움이 있다.) 나에게는 맞지 않은 툴. 모든 기능이 있지만 특화되어있지 않다 보니 어설픈 느낌이라고 할까.. 다만 노트는 확실히 예쁘게 꾸밀 수 있어 굳이 하자면 Dairy나 Journal 형태의 노트들이 어울릴 것 같다.
