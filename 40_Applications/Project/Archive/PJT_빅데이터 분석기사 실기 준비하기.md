## 개요 
시험일은 12/2(토)이며, 시험유형이 변경되어서 필답형은 없어지고 작업형1~3형으로 변경되었다.
공부 순서는 '빅데이터 분석 이론 이해' > '작업형 실기 기본기/실전 다지기' 순으로 진행예정이며, 이론 공부는 최대한 간결하고 빠르게 진행하고 실기 실전연습을 확실히 하는 것을 목표로 하자.


## 내용
#### 1. 제7회 시험일정 및 유형분석
   - 시험유형 : [[빅데이터 분석기사 실기 유형 분석]]
   - 시험일정
	   - 접수기간 : 10.30~11.3
	   - 수험표발급 : 11.17
	   - 시험일 : 12.2
	   - 사전점수공개 : 12.15~19
	   - 결과발표 : 12.22

#### 2. 빅데이터 분석에 대한 이해
   : 필기공부를 한지 1년이 지난 상태에서 다시 한번 복습이 필요할 것 같다. 전체적인 데이터 분석에 대한 흐름에 대해서 한번은 복습하고 실기 공부를 준비해보자. 

1) 빅데이터 분석기사 필기책 요약본 
   - 전체 내용들에 대한 목차와 주요 이론들을 목록화
   - 추가적으로 필요한 이론들은 옵시디언에 노트화 시켜서 세부화하기
2) 빅데이터 분석기사 실기책
   - 필답형 빠르게 훑어보면서 필기내용과 Cross Checking 하기
   - 작업형도 역시 빠르게 훑어보면서 전반적인 적용 흐름 파악하기 (실습은 아래 내용들로 하기)

#### 3. 빅데이터 분석 작업형 실기 준비

1) Big Data Certification 기본기 다지기 (https://www.kaggle.com/datasets/agileteam/bigdatacertificationkr)
   - 파이썬, 판다스 기본 문법 전체 숙달하기
   - 작업형 1~3유형 예시/기출문제 풀기
2) Data Manim에서 실전 다지기 (https://www.datamanim.com/dataset/03_dataq/re.html)
   - 데이터 전처리 100문제 풀기
   - 작업형 1~3유형 연습문제
   - 구름 환경 연습하기
