## 개요
2개의 프로젝트를 나눠 수행한다. 약 남은기간은 2달. 적당한 비율로 시간을 분배하여 공부 진행
여러 분산된 지식들을 하나로 결합하여, 나만의 지식을 구축하여 실기 자격증도 취득하고 현업에서도 활용할 수 있는 지식을 습득하는데 목표로 한다.

## 내용

#### [[PJT_빅데이터 분석기사 실기 준비하기]]
   : 시험일정 및 유형 등을 파악하여 공부를 시작할 수 있도록 준비


#### [[PJT_빅데이터 분석 이론 이해하기]]
   : 전체적인 데이터 분석에 대한 흐름에 대해서 한번은 복습, 암기는 필요없기 때문이 이해 위주의 빠른 진행 필요
- Due Date : 9/24(일)~10/15(일)
- Target
	- 빅데이터 분석기사 필기책 요약본 정독, 개념 목록화
		- 전체 내용들에 대한 목차와 주요 이론들을 목록화
		- 추가적으로 필요한 이론들은 옵시디언에 노트화 시켜서 세부화하기
	- 빅데이터 분석기사 실기책 Cross Checking
		- 필답형 빠르게 훑어보면서 필기내용과 Cross Checking 하기
		- 작업형도 역시 빠르게 훑어보면서 전반적인 적용 흐름 파악하기 (실습은 아래 내용들로 하기)


#### [[PJT_빅데이터 분석 수행 (Python)]]
   : 파이썬을 활용한 데이터 분석 실습 및 숙달
- Due Date : 10/16(월)~12/1(금)
- Target
	- 빅데이터 분석기사 실기책 기준 뼈대 잡기
	- Big Data Certification 기본기 다지기 (https://www.kaggle.com/datasets/agileteam/bigdatacertificationkr)
		- 파이썬, 판다스 기본 문법 전체 숙달하기

		- 작업형 1~3유형 예시/기출문제 풀기
	- Data Manim에서 실전 다지기 (https://www.datamanim.com/dataset/03_dataq/re.html)
		- 데이터 전처리 100문제 풀기
		- 작업형 1~3유형 연습문제
		- 구름 환경 연습하기
